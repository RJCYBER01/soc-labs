# SOC Labs 🔐

This repository documents my hands‑on cybersecurity labs focused on SOC operations, log analysis, alert triage, and threat hunting.

## 🧠 Objective
To build practical experience in detecting, analyzing, and responding to security incidents using real‑world tools and data.

## 🧰 Tools Used
- Splunk / ELK Stack
- Wireshark
- Security Onion
- Azure Sentinel
- Sysmon
- Python / PowerShell for automation

## 🧪 Labs
| Lab | Topic | Description |
|-----|--------|-------------|
| Lab 01 | Log Analysis | Investigating suspicious login activity |
| Lab 02 | Alert Triage | Reviewing alerts and prioritizing incidents |
| Lab 03 | Threat Hunting | Searching for indicators of compromise |

## 📚 Learning Goals
- Strengthen SOC investigation skills  
- Practice log correlation and alert validation  
- Document findings clearly and professionally
